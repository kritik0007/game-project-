A fast-paced 2D fighting game built using HTML5 Canvas, JavaScript, and Node.js.
Player vs AI combat with combo mechanics, knockback physics, and backend score saving.

This project demonstrates core game loop architecture, collision detection, state management, and frontend-backend integration.



🎮 Gameplay Overview

Side-view 2D fighting

Player vs AI opponent

Combo-based attack system

Knockback physics

Health bars

Game over overlay

Backend score saving (Node.js + Express)


Key	Action
A	Move Left
D	Move Right
W	Jump
Space	Attack (Combo System)


⚔ Features
🧠 Combat Mechanics

3-hit combo chain

Damage scaling per combo step

Attack cooldown system

Hit detection using bounding box collision

Knockback proportional to combo strength

🎨 Visual Effects

Dark gradient battlefield

Hit flash feedback

Attack hitbox debug visualization

Overlay-based game over screen

🌐 Backend Integration

Node.js + Express server

Score saving via REST API

CORS enabled

Ready for database upgrade
